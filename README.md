# NPBCENTRAL20052024
プロ野球セ・リーグ順位推移ダッシュボード（2005年～2024年）
当リポジトリは、Power Query / Power BI / DAX を用いて  
時系列データの可視化・分析を行ったサンプルダッシュボードです。

私は自社開発・SES現場にて SQL / Excel / VBAを中心に業務してきましたが、  
BI案件（既存レポート改修・運用フェーズ）への参画に備え、
業務で使用したPowerQueryから拡張してPower BIを学習・アウトプットしたものです。

## ダッシュボード概要

- 折れ線グラフによる順位推移の可視化
- スライサーによるチーム（部署）選択
- 複数集計指標の設置（平均・最大）
- 前年／前期間（2020年～2021年）との差分表示

## 使用技術

- Power BI Desktop
- Power Query（データ整形・前処理）
- DAX
  - AVERAGE
  - MAX
  - CALCULATE
  - FILTER
  - 差分計算（Diff）

## 業務での活用想定

業務上の実データは難しいと考え、イメージしやすい公開データであるプロ野球セ・リーグ　2005年～2024年の順位データを
使用しています。以下のような業務データへの置き換えを想定しています。

- 営業部署ごとのKPI推移
- 月次／四半期の実績比較
- 部署別・担当者別の成績可視化
- 既存ExcelレポートのPower BI化

## 分析テーマについて

前年差分比較の期間を2020年～2021年とした理由は、  
東京ヤクルトスワローズが **最下位から優勝** しているためです。

改善施策の実施、競合環境の変化など、  
業務における数値改善要因の分析に置き換えやすいケースであり、  BIレポートとしての活用イメージがしやすいと考えました。
